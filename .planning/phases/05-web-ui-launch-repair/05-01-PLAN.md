| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-05-02 | D | persisted UI startup path | mitigate | Add an end-to-end subprocess regression that proves enable, persisted relaunch, and disable behavior against the real binary so silent launch failures are caught before release. |
| T-05-03 | I | HTTP UI bind behavior | mitigate | Preserve the existing `cbm_http_server_new(ui_cfg.ui_port)` path and the server's localhost-only binding contract; do not add alternate host binding or new remote-facing UI modes while repairing startup. |
</threat_model>
