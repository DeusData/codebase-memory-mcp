## Trust Boundaries

| Boundary | Description | Data Crossing |
|----------|-------------|---------------|
| untrusted `.gd` source -> extractor AST walk | Arbitrary repository code is parsed and converted into indexed definitions/properties | source code tokens, symbol metadata |
